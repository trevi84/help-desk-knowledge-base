# Computer Won't Boot

## Symptoms

- Black screen
- No Windows logo
- Error messages

## Troubleshooting

1. Verify power cable.
2. Check monitor connection.
3. Remove USB devices.
4. Restart computer.
5. Boot into Safe Mode.
6. Run Startup Repair.
7. Review Event Viewer after startup.

## Resolution

The system boots successfully after Startup Repair.
