
# Password Reset

## Problem

A user cannot log into their Windows account because they forgot their password.

## Troubleshooting Steps

1. Verify the user's identity.
2. Open Active Directory Users and Computers.
3. Locate the user account.
4. Right-click the account.
5. Select Reset Password.
6. Enter a temporary password.
7. Require password change at next login.
8. Unlock the account if necessary.

## Resolution

The user successfully signs in using the temporary password and creates a new password.
