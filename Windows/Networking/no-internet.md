
# No Internet Connection

## Symptoms

- Unable to browse websites
- Network icon shows disconnected

## Troubleshooting

1. Check Ethernet/Wi-Fi.
2. Restart router.
3. Run:

ipconfig /release

ipconfig /renew

ipconfig /flushdns

4. Ping 8.8.8.8
5. Ping google.com
6. Verify DNS settings.

## Resolution

Internet connectivity restored after renewing the IP address.
